---

## 🛠️ Tech Stack & Features
This portfolio is built using:
- **Framework:** [Next.js](https://nextjs.org/) (v16 App Router) & [React](https://react.dev/) (v19)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (v4) for highly customizable, responsive layout design
- **Animations:** [Framer Motion](https://www.framer.com/motion/) for fluid page transitions and interactive micro-animations
- **Icons:** [Lucide React](https://lucide.dev/) & Devicons
- **Deployment:** Hosted on [Vercel](https://vercel.com/)

## 📂 Key Projects Showcased
1. **Face Recognition Attendance Management System (FRAMS)**
   - An AI-powered attendance system that automates marking using real-time face recognition. Uses OpenCV, LBPH algorithm, and MySQL.
2. **AI News Research Tool**

## 🛠️ Local Development

### 1. Clone the Repository
```bash
git clone https://github.com/siddharthprajapati05/Sidd_Portfolio.git
cd Sidd_Portfolio/portfolio
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Development Server
```bash
npm run dev
```
The server will start on [http://localhost:3001](http://localhost:3001).

### 4. Build for Production
```bash
npm run build
```
